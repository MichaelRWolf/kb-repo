Before buying:

  • Ask about non-standard model number MUHQ2LLA because looking it up on the Apple site does not give any specifications: https://support.apple.com/specs/MUHQ2LLA

  • See if you can find a different model number and paste it in above browser link

  • Here are specs for your current computer:

    • Model Number MC374LL/A - https://support.apple.com/specs/MC374LL

    • Model Number MC375LL/A - https://support.apple.com/specs/MC375LL

    • Serial Number W8047D4TATM – https://support.apple.com/specs/W8047D4TATM

  • Square Trade warranty - https://www.squaretrade.com/laptop-warranty

    • 2 year - $230

    • 3 year - $329

  • Apple Care (PDF…) is probably $250

Future Computer

https://www.apple.com/mac/compare/

Steve recommends Costco + SquareTrade warranty.

Avoid Optical drive. It indicates vintage computers. If we need one, we can buy a new one from Apple for $80 - https://www.apple.com/shop/product/MD564LL/A/apple-usb-superdrive, or a lesser brand for $30-40.

Go for Retina display. It’s better. It indicates a newer model.

You probably won’t use Touch Bar, though it can’t help.
